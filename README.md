# 🌐 Personal Portfolio Website (Flask)

A modern and responsive **Personal Portfolio Website** built using **Flask (Python)** for backend and **HTML, CSS, JavaScript** for frontend.
This portfolio showcases my projects, skills, and provides a contact form to connect with me directly via email.

---

## 🚀 Features

* 🏠 Home Page with introduction
* 👨‍💻 About Me section
* 📂 Projects showcase page
* 📩 Contact form with email integration (Gmail SMTP)
* 🎨 Responsive UI with modern design
* ⚡ Smooth animations and interactive elements

---

## 🛠️ Tech Stack

* **Backend:** Python (Flask)
* **Frontend:** HTML5, CSS3, JavaScript
* **Styling:** Bootstrap, Custom CSS
* **Email Service:** SMTP (Gmail)
* **Version Control:** Git & GitHub

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/your-username/your-repo-name.git
```

2. Navigate to the project folder:

```
cd your-repo-name
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Create a `.env` file and add:

```
EMAIL=yourgmail@gmail.com
PASSWORD=your_app_password
```

5. Run the Flask app:

```
python app.py
```

6. Open in browser:

```
http://127.0.0.1:5000/
```
